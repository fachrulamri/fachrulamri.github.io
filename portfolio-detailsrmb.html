Rekomendasi Mobil Bekas

Website yang membahas tentang rekomendasi mobil bekas. Dalam website ini terdapat metode database fuzzy model tahani dimana dapat memberikan hasil rekomendasi mobil bekas berdasarkan input kriteria mobil bekas yang bersifat samar (tidak jelas nilainya). Website ini dibuat menggunakan bahasa pemrograman php dengan framework laravel dan template boostrap.
